\\([A, B] = 0\\) より、

\begin{eqnarray\*}
&\phantom{\implies}& [A, B] = 0 \\\\\\
&\implies& AB - BA = 0 \\\\\\
&\implies& AB = BA
\end{eqnarray\*}

\\(\\{A, B\\} = 0\\) より、

\begin{eqnarray\*}
&\phantom{\implies}& \\{A, B\\} = 0 \\\\\\
&\implies& AB + BA = 0 \\\\\\
&\implies& 2AB = 0\ (\because AB = BA) \\\\\\
&\implies& AB = 0
\end{eqnarray\*}

\\(A\\) は可逆だから \\(A^{-1}\\) が存在し、

\begin{eqnarray\*}
&\implies& A^{-1}AB = A^{-1}0 \\\\\\
&\implies& B = 0
\end{eqnarray\*}
