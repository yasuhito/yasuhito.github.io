エバーグリーンノートのタイトル付けは API 設計によく似ている。

-   単機能にする (例: `Object#inspect`)
-   キャッチーな名詞にする (例: "囚人のジレンマ"、"割れ窓理論")
-   一発で決まらない時は、時間とともに改善 (リファクタリング)
