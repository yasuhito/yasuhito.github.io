QCQI/演習 2.43

 Apr 20, 2021

\[\sigma_j \sigma_k = \frac{[\sigma_j, \sigma_k] + \{\sigma_j, \sigma_k\}}{2}\]

ここで QCQI/演習 2.40 より、

\[[\sigma_j, \sigma_k] = 2i\sum_{l=1}^{3}\epsilon_{jkl}\sigma_l\]

また QCQI/演習 2.41 より、

\(j \neq k\) のとき

\[\{\sigma_j, \sigma_k\} = 0\]

\(j = k\) のとき、

\[\{\sigma_j, \sigma_k\} = \sigma_j\sigma_k + \sigma_k\sigma_j = 2I\]

よって

\begin{eqnarray*} \frac{[\sigma_j, \sigma_k] + \{\sigma_j, \sigma_k\}}{2} &=& \frac{1}{2}\left(2i\sum_{l=1}^{3}\epsilon_{jkl}\sigma_l + 2\delta_{jk}I\right) \\\
&=& \delta_{jk}I + i\sum_{l=1}^{3}\epsilon_{jkl}\sigma_l \end{eqnarray*}
