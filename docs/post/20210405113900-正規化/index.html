正規化

 Apr 5, 2021

ベクトル \(|v\rangle\) について、もしノルム \(\|v\rangle\| = 1\) ならば、\(|v\rangle\) は正規化 (または規格化) されているという。ゼロでないベクトル \(|v\rangle\) について \(|v\rangle / \||v\rangle\|\) は \(|v\rangle\) を正規化した形。
