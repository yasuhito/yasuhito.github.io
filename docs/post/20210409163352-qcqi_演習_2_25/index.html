QCQI/演習 2.25

 Apr 9, 2021

転置共役の定義と \((A^{\dagger})^{\dagger} = A\) (QCQI/演習 2.15) より、

\[(|v\rangle, A^{\dagger}A|v\rangle) = (|v\rangle, A^{\dagger}(A|v\rangle)) = (A|v\rangle, A|v\rangle)\]

ここで内積の正値性より、

\begin{align*} &(A|v\rangle, A|v\rangle) \geq 0 \\\
\Rightarrow\ &(|v\rangle, A^{\dagger}A|v\rangle) \geq 0 \end{align*}

よって、\(A^{\dagger}A\) は正の演算子。
