基底状態

 Feb 24, 2021

\(|0\rangle\) と \(|1\rangle\) は基底状態と呼ばれ、古典コンピュータのビット 0 と 1 に対応する。数学的には、基底状態は状態ベクトル \(|\psi\rangle\) の正規直交基底を形成するベクトルであり、それぞれ次の列ベクトルで定義される。

\begin{equation} |0\rangle := \begin{bmatrix} 1 \\\
0 \end{bmatrix}, |1\rangle := \begin{bmatrix} 0 \\\
1 \end{bmatrix} \end{equation}

よって 1 量子ビットの状態 \(|\psi\rangle\) は、2次元複素ベクトル空間の単位ベクトルとして表現できる。

\begin{equation} |\psi\rangle = \begin{bmatrix} \alpha \\\
\beta \end{bmatrix} \end{equation}
