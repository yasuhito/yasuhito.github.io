演算子の合成

 Apr 1, 2021

\(V\), \(W\), および \(X\) がベクトル空間であり、\(A: V \rightarrow W\) と \(B: W \rightarrow X\) が演算子であるとする。このとき \(A\) と \(B\) の合成を記号 \(BA\) で表し、\((BA)(|v\rangle) \equiv B(A|v\rangle)\) で定義する。

演算子と行列は等価なので、\((BA)(|v\rangle)\) は括弧を省略して \(BA|v\rangle\) と書ける。
