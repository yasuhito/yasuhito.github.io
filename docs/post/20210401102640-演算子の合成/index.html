演算子の合成

 Apr 1, 2021

\(V\), \(W\), および \(X\) がベクトル空間であり、\(A: V \rightarrow W\) と \(B: W \rightarrow X\) が演算子であるとする。このとき \(A\) と \(B\) の合成を記号 \(BA\) で表し、

\[(BA)(\ket{v}) \equiv B(A\ket{v})\]

で定義する。
