線形従属

 Mar 31, 2021

\(0\) でないベクトル \(|v_1\rangle, \ldots, |v_n\rangle\) について、少なくとも 1 つの \(i\) に対して \(a_i \neq 0\) であり、

\[a_1|v_1\rangle + a_2|v_2\rangle + \ldots + a_n|v_n\rangle = 0\]

を満たす複素数の集合 \(a_1, \ldots, a_n\) が存在するならば、ベクトル \(|v_1\rangle, \ldots, |v_n\rangle\) は線形従属であるという。
