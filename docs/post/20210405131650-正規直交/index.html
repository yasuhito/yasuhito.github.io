正規直交

 Apr 5, 2021

インデックス \(i\) を持つベクトルの集合 \(|i\rangle\) の各々が単位ベクトルであり、その集合のうち互いに異なるベクトルが直交しているとき、つまりインデックスの集合から選んだ \(i\) と \(j\) に対して \(\langle i|j \rangle = \delta_{ij}\) のとき、集合 \(|i\rangle\) は正規直交であるという。
