QCQI/演習 2.68

 May 5, 2021

\(\ket{a} = a_1\ket{0} + a_2{\ket{1}}\), \(\ket{b} = b_1\ket{0} + b_2{\ket{1}}\) とおき \(\ket{a}\ket{b}\) を計算すると、

\begin{eqnarray*} \ket{a}\ket{b} &=& (a_1\ket{0} + a_2{\ket{1}})(b_1\ket{0} + b_2{\ket{1}}) \\\
&=& a_1b_1\ket{00} + a_1b_2\ket{01} + a_2b_1\ket{10} + a_2b_2\ket{11} \end{eqnarray*}

これが \(\ket{\psi} = (\ket{00} + \ket{11})/\sqrt{2}\) と等しいとすると、

\[a_1b_1 = 1/\sqrt{2},\ a_1b_2 = 0,\ a_2b_1 = 0,\ a_2b_2 = 1/\sqrt{2}\]

これを満たす \(a_1, a_2, b_1, b_2\) はないので、\(\ket{\psi} \neq \ket{a}\ket{b}\)
