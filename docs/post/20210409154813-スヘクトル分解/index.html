スペクトル分解

 Apr 9, 2021

ベクトル空間 \(V\) 上の任意の正規演算子 \(M\) は \(V\) のある正規直交基底に関して対角である。逆に任意の対角化可能な演算子は正規である。

外積表現では、この定理は \(M\) が \(M = \sum_i \lambda_i |i\rangle\langle i|\) と書けることを意味する (\(\lambda_i\) は \(M\) の固有値、各 \(|i\rangle\) は固有値 \(\lambda_i\) に対応する \(M\) の固有ベクトルであり、\(V\) の正規直交基底を形成する)。
