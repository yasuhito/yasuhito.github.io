QCQI/演習 2.7

 Apr 5, 2021

\(\ket{w}\) と \(\ket{v}\) の内積を計算すると、

\begin{equation} \braket{w|v} = \begin{bmatrix} 1 & 1 \end{bmatrix}^{\ast}\begin{bmatrix} 1 \newline -1 \end{bmatrix} = 1\times1 + 1 \times (-1) = 0 \end{equation}

よって \(\ket{w}\) と \(\ket{v}\) は直交している。

\(\|\ket{w}\| = \sqrt{\braket{w|w}} = \sqrt{2},\ \|\ket{v}\| = \sqrt{\braket{v|v}} = \sqrt{2}\) なので、それぞれを正規化すると、

\begin{equation*} \frac{\ket{w}}{\|\ket{w}\|} = \frac{1}{\sqrt{2}}\ket{w},\ \frac{\ket{v}}{\|\ket{v}\|} = \frac{1}{\sqrt{2}}\ket{v} \end{equation*}
