ヒルベルト空間

 Apr 5, 2021

ベクトル空間のうち内積を備えており (= 内積空間)、かつベクトルの要素が実数か複素数の範囲であり連続 (数学の言葉で「完備性を持つ」と言う) つまり微分などの計算もできるベクトル空間のこと。平たく言うと和、スカラー倍、内積、微分などの「ふつうの計算」には困らないベクトル空間をヒルベルト空間と呼ぶ。

量子コンピュータではふつう、有限次元の複素ベクトル空間についてのみ扱う。この場合「内積空間 = ヒルベルト空間」。なので、ヒルベルト空間と聞けばそれは内積空間のことだと思ってよい。
