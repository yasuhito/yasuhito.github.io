QCQI/演習 2.44

 Apr 20, 2021

\([A, B] = 0\) より、

\begin{eqnarray*} &\phantom{\Rightarrow}& [A, B] = 0 \\\
&\Rightarrow& AB - BA = 0 \\\
&\Rightarrow& AB = BA \end{eqnarray*}

\(\{A, B\} = 0\) より、

\begin{eqnarray*} &\phantom{\Rightarrow}& \{A, B\} = 0 \\\
&\Rightarrow& AB + BA = 0 \\\
&\Rightarrow& 2AB = 0\ (\because AB = BA) \\\
&\Rightarrow& AB = 0 \end{eqnarray*}

\(A\) は可逆だから \(A^{-1}\) が存在し、

\begin{eqnarray*} &\Rightarrow& A^{-1}AB = A^{-1}0 \\\
&\Rightarrow& B = 0 \end{eqnarray*}
