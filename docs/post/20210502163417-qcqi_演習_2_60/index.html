QCQI/演習 2.60

 May 2, 2021

QCQI/演習 2.35 の計算より、\(\vec{v}\cdot\vec{\sigma}\) の固有値は \(\pm{1}\)

固有値 \(\pm{1}\) それぞれの固有ベクトルを \(\ket{\lambda_{+}}, \ket{\lambda_{-}}\) とすると、

\(\vec{v}\cdot\vec{\sigma}\) のスペクトル分解により

\[\vec{v}\cdot\vec{\sigma} = \ket{\lambda_{+}}\bra{\lambda_{+}} - \ket{\lambda_{-}}\bra{\lambda_{-}}\]

完全性関係より、

\[I = \ket{\lambda_{+}}\bra{\lambda_{+}} + \ket{\lambda_{-}}\bra{\lambda_{-}}\]

よって

\[P_{+} = \ket{\lambda_{+}}\bra{\lambda_{+}} = \frac{I + \vec{v}\cdot\vec{\sigma}}{2}\] \[P_{-} = \ket{\lambda_{-}}\bra{\lambda_{-}} = \frac{I - \vec{v}\cdot\vec{\sigma}}{2}\]
