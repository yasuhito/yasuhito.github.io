ユニタリー演算子の外積表現

 Apr 8, 2021

\(|v_i\rangle\) は任意の正規直交基底の集合であるとする。\(|w_i\rangle \equiv U|v_i\rangle\) を定義すると、ユニタリー演算子は内積を保存するので \(|w_i\rangle\) も正規直交基底の集合である。したがって \(U = \sum_i |w_i\rangle\langle v_i|\) が成り立つ。逆にもし \(|v\rangle\) と \(|w\rangle\) が任意の 2 つの直交基底ならば、\(U \equiv \sum_i |w_i\rangle\langle v_i|\) で定義される演算子 \(U\) はユニタリー演算子であることが容易に確認できる。
