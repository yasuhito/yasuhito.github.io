テンソル積の双線形性

 Apr 12, 2021

\(V\) と \(W\) をそれぞれヒルベルト空間とすると、テンソル積は次の基本的性質を満たし、これを双線形性と呼ぶ。

  1. 任意のスカラー \(z\), \(V\) の要素 \(|v\rangle\), \(W\) の要素 \(|w\rangle\) に対して、 \[z(|v\rangle \otimes |w\rangle) = (z|v\rangle) \otimes |w\rangle = |v\rangle \otimes (z|w\rangle)\]

  2. \(V\) の任意の要素 \(|v_1\rangle\) と \(|v_2\rangle\), \(W\) の要素 \(|w\rangle\) に対して、 \[(|v_1\rangle + |v_2\rangle) \otimes |w\rangle = |v_1\rangle \otimes |w\rangle + |v_2\rangle \otimes |w\rangle\]

\[|v\rangle \otimes (|w_1\rangle + |w_2\rangle) = |v\rangle \otimes |w_1\rangle + |v\rangle \otimes |w_2\rangle\]
