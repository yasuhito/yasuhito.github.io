閉じた量子系の時間発展はユニタリー変換で記述できる

 Apr 22, 2021

閉じた量子系について、時刻 \(t_1\) における系の状態 \(\ket{\psi}\) は、時刻 \(t_2\) における系の状態 \(\ket{\psi^{\prime}}\) と時刻 \(t_1\) と \(t_2\) だけに依存するユニタリー演算子 \(U\) によって関連付けられる。

\[\ket{\psi^{\prime}} = U\ket{\psi}\]
