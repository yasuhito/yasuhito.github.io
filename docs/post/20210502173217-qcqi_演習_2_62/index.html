QCQI/演習 2.62

 May 2, 2021

仮に POVM 要素が測定演算子と一致していたとすると、\(E_m = M_m^{\dagger}M_m = M_m\) となる。\(E_m = M_m\) で \(E_m\) はエルミート演算子なので、\(M_m^{\dagger}M_m = M_m^2 = M_m\) を満たす必要がある。これは \(M_m\) が射影測定であるための条件であり、射影演算子はべき等性 (\(P^2 = P\)) を満たすので、この測定は射影測定であることを示している。
