行列は演算子とみなせる

 Apr 1, 2021

要素 \(A_{ij}\) を持つ \(m \times n\) の複素行列 \(A\) は、行列 \(A\) と \(\mathcal{C}^n\) とのベクトルとの行列積により、ベクトル空間 \(\mathcal{C}^n\) からベクトル空間 \(\mathcal{C}^m\) に移すので、行列 \(A\) は演算子である。

正確には行列 \(A\) が演算子であるということは

\[A\left(\sum_i a_i|v_i\rangle\right) = \sum_i a_i A(|v_i\rangle)\]

が行列 \(A\) と列ベクトルの行列積で成り立つことを意味し、行列の性質からこれは明らかに正しい。
