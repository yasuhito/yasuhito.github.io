演算子

 Mar 30, 2021

線形演算子を略して演算子 (operator) と呼ぶ。量子計算や量子情報など主に有限次元のヒルベルト空間を扱う場合演算子と行列は等価なので、「線形」を略して単に「演算子」と呼ぶことが多い。

この演算子の正体は行列ということと、ディラック記法での計算のしやすさから、\(A(\ket{v})\) を \(A\ket{v}\), 演算子の合成 \((BA)(\ket{v})\) を \(BA\ket{v}\) などとカッコを省略して書くのが普通。
