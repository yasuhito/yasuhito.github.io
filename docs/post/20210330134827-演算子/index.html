演算子

 Mar 30, 2021

量子論では、状態ベクトルに対する線形変換のことを演算子またはオペレータ (operator) と呼ぶ。数学の分野では同じ用語 operator を作用素と訳すので注意。

演算子は、入力に対して線形な任意の関数 A として次のように定義できる。

\[A(\sum_i a_i|v_i\rangle) = \sum_i a_i A(|v_i\rangle)\]
