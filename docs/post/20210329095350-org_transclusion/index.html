org-transclusion

 Mar 29, 2021

Org Mode で Roam Research 的なブロック参照を行うためのパッケージ。

Org-Roam で書いたノートの一部を、他のノートや Doom Emacs の config.org から参照したくなったので導入。

設定

~/.doom.d/packages.el の設定

(package! org-transclusion
  :recipe (:host github
           :repo "nobiot/org-transclusion"
           :branch "main"))

~/.doom.d/config.el の設定

(use-package! org-transclusion
  :hook (org-roam-mode . org-transclusion-mode))
