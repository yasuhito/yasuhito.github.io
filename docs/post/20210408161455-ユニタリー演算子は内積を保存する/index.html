ユニタリー演算子は内積を保存する

 Apr 8, 2021

幾何学的にユニタリー演算子はベクトル間の内積を保存する。これを見るために \(|v\rangle\) と \(|w\rangle\) を任意の 2 つのベクトルとすると、\(U|v\rangle\) と \(U|w\rangle\) の内積は \(|v\rangle\) と \(|w\rangle\) の内積に等しい。

\begin{equation} (U|v\rangle, U|w\rangle) = \langle v|U^{\dagger}U|w\rangle = \langle v|I|w\rangle = \langle v|w\rangle \end{equation}
