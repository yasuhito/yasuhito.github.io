固有値

 Apr 6, 2021

ベクトル空間における演算子 \(A\) について、\(A|v\rangle = v|v\rangle\) を満たす固有ベクトル \(|v\rangle\) に対応する複素数 \(v\) を \(A\) の固有値と呼ぶ。
