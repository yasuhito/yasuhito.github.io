エルミート共役

 Mar 29, 2021

\(m \times n\) 複素行列 \(A\) に対して、\(A\) の転置および複素共役をとって得られる \(n \times m\) 行列をエルミート共役と呼ぶ。随伴行列、エルミート随伴とも呼ばれる。

量子論ではエルミート共役は \(A^{\dagger}\) と書き、\(A^{\dagger} = (A^T)^{\ast}\) である。
