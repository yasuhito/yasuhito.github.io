エルミート演算子

 Apr 8, 2021

演算子 \(A\) の転置共役が元の演算子 \(A\) に等しいとき、つまり

\[A^{\dagger} = A\]

なとき、その演算子をエルミート演算子または自己共役 (自己随伴) 演算子であるという。

エルミート演算子は正規なのでスペクトル分解できる。
