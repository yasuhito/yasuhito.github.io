線形結合

 Feb 23, 2021

和演算とスカラー倍演算を組合わせることで、ベクトル \(|v_1\rangle\) と \(|v_2\rangle\) の線形結合を作ることができる。\(|v_1\rangle\) に \(c\) を掛け、\(|v_2\rangle\) に \(d\) を掛け、さらに和 \(c|v_1\rangle + d|v_2\rangle\) を求める。これが \(|v_1\rangle\) と \(|v_2\rangle\) の線形結合。
