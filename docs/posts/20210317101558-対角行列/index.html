対角行列

2021-03-17

\(n\) 次正方行列で、その対角線上にない成分がすべて \(0\) である行列のこと。

\[\begin{bmatrix} 1 & 0 & 0 \\\
0 & 2 & 0 \\\
0 & 0 & -1 \end{bmatrix}\]

注意点として、行列の対角線は右下がりのもののみを言う。右上がりのものは対角線とは呼ばない。
