スカラー行列

2021-03-17

単位行列の定数倍をスカラー行列という。

たとえば、

\[\begin{bmatrix} \lambda & 0 & 0 \\\
0 & \lambda & 0 \\\
0 & 0 & \lambda \end{bmatrix}\]

はスカラー行列である。
